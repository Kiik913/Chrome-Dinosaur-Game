# Chrome Dino Game

A standalone version of the famous Chrome Dino Game built using HTML, CSS, and JavaScript.

## Features

- Classic Dino gameplay
- Jump and dodge obstacles
- Score tracking
- Responsive design
- Works offline
- Lightweight and fast

## Controls

| Key | Action |
|------|---------|
| Space | Jump |
| Up Arrow | Jump |
| Down Arrow | Duck |
| Enter | Restart |

## Installation

1. Clone the repository:

```bash
git clone https://github.com/Kiik913/Chrome-Dinosaur-Game.git
```

2. Open the project folder.

3. Run `Chrome Dinosaur Game.html` in your browser.
